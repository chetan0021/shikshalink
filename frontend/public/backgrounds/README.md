# Optional hero background

To get a soft, editorial look similar to minimalist studio sites, add **one** of these files:

- `hero-bg.webp` — recommended (light grain, abstract, low contrast)
- `hero-bg.gif` — subtle loop only (keep file size small)

Place the file in this folder: `public/backgrounds/`

The hero will detect it automatically. If no file is present, the page uses a clean paper background only.

**Tips**

- Resolution: ~1920×1080 or larger; we display it faintly (`opacity` + `mix-blend-multiply`).
- Avoid busy photos; soft texture or abstract motion works best.
